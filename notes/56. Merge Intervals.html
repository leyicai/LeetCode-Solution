56. Merge Intervals

1 Sort starts and ends seperately

Fastest.
The latter one’s start must > previous one’s end

1.public int[][] merge(int[][] intervals) {

2.    int n = intervals.length;

3.    int[] starts = new int[n];

4.    int[] ends = new int[n];

5.    for (int i = 0; i < n; i++) {

6.        starts[i] = intervals[i][0];

7.        ends[i] = intervals[i][1];

8.    }

9.    Arrays.sort(starts);

10.    Arrays.sort(ends);

11.    List<int[]> res = new ArrayList<>();

12.    for (int i = 0, j = 0; i < n; i++) {  // j is the start of the new interval

13.        if (i == n - 1 || ends[i] < starts[i + 1]) {

14.            // current end(i) < next start, a new interval

15.            int[] newInterval = new int[2];

16.            newInterval[0] = starts[j];

17.            newInterval[1] = ends[i];

18.            res.add(newInterval);

19.            j = i + 1;

20.        }

21.    }

22.    return res.toArray(int[res.size()][]);

2 Update new interval’s end

  1. Sort the intervals by starts

  2. Compare current end with next start, if overlapped, update the end with the current max end.

  3. If not overlapped, which means a new interval start was found, we add it into result
