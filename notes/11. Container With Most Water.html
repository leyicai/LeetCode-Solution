11. Container With Most Water


move the pointer with smaller height

2.    public int maxArea(int[] height) {

3.        int area = 0, lo = 0, hi = height.length - 1;

4.        while (lo < hi) {

5.            int edge = Math.min(height[lo], height[hi]);

6.            area = Math.max(area, edge * (hi - lo));

7.            if (height[lo] < height[hi])

9.            else

10.                hi--;

11.        }

12.        return area;
