152. Maximum Product Subarray

O(n^2) time and O(1) space

  • for each num[i], compute product of nums[i:j]

1.public int maxProduct(int[] nums) {

2.    int res = nums[0];

3.    int n = nums.length;

4.    for(int i = 0; i < n; i++){

5.        int prev = 1, curr;

6.        for (int j = i; j < n; j++) {

7.            curr = prev * nums[j];

8.            res = Math.max(res, curr);

9.            prev = curr;

10.        }

11.    }

12.    return res;

13.}

O(n) time and O(1) space

  • max and min are the max/min product end with nums[i]

  • Due to negative number and property of multiply, we need max and min ends at i-1

1.public int maxProduct(int[] nums) {

2.    int res = nums[0];

3.    int max = nums[0], min = nums[0];

4.    for (int i = 1; i < nums.length; i++) {

5.        int prevMax = max;

6.        max = Math.max(nums[i], Math.max(prevMax * nums[i], min * nums[i]));

7.        min = Math.min(nums[i], Math.min(min * nums[i], prevMax * nums[i]));

8.        res = Math.max(res, max);

9.    }

10.    return res;

11.}
